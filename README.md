# ♟️ Шахматы

Классическая игра **«Шахматы»**, разработанная на **Python** с использованием библиотеки **Tkinter**.

Проект представляет собой настольное приложение с графическим интерфейсом, в котором реализованы основные правила шахмат, включая специальные ходы и завершение партии.

---

## 🚀 Запуск

### Способ 1. Запуск готовой сборки

Если вы не хотите устанавливать Python, откройте папку **dist** и запустите файл:

```
dist/chess.exe
```

### Способ 2. Запуск из исходного кода

Убедитесь, что установлен **Python 3.10** или новее.

После этого выполните:

```bash
python chess.py
```

> Tkinter входит в стандартную библиотеку Python и не требует отдельной установки.

## 🎮 Возможности

- ♟️ Полноценная шахматная доска 8×8
- 👆 Управление с помощью мыши
- 👑 Короткая и длинная рокировка
- ♟️ Превращение пешки
- ⚔️ Взятие на проходе
- 🚨 Проверка шаха, мата и пата
- 🎨 Несколько цветовых тем доски
- 📖 Встроенные правила игры
- 🖥️ Графический интерфейс на Tkinter
